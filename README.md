# Anyhenge

See what dates the sunset aligns with any street grid on Earth.

https://anyhenge.ziki.boo

## Background

[Manhattanhenge](https://en.wikipedia.org/wiki/Manhattanhenge) is the phenomenon where Manhattan's street grid aligns with the setting sun twice a year. The same thing happens on any east-west grid anywhere in the world — it just doesn't have a name.

Anyhenge lets you find that moment for any street, anywhere.

## How it works

Pan the map to any street grid, then drag the date slider to find the day when the sunset azimuth line aligns with the streets.

Built with [Leaflet](https://leafletjs.com/) and [SunCalc.js](https://github.com/mourner/suncalc). Single HTML file, no build step, deployed on [Cloudflare Workers](https://workers.cloudflare.com/).
